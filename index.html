Analyzing Indonesia's Reading Culture Through Data

Using the Tingkat Kegemaran Membaca (TGM) dataset, I explored reading interest levels across Indonesian provinces. This project combined SQL for data processing and Tableau for visualization, resulting in an interactive dashboard showcasing trends and actionable insights.

Key Highlights:

  • Year-over-Year Growth: Identified West Sulawesi as a leader with an 18.02% increase in 2022, reflecting successful literacy programs.
  • Regional Comparisons: Yogyakarta ranked highest (73.27 TGM score), while Papua faced challenges (60.58), emphasizing disparities.
  • Metrics Analyzed: Reading frequency and daily reading duration by province, uncovering patterns for targeted interventions.

COVID-19 Global Dashboard: Insights from 2020-April 2021

I explored the global impact of COVID-19 by combining SQL exploration and Tableau visualization. The goal was to create an interactive dashboard that simplifies complex data and provides actionable insights.

Key Analyses:

  • SQL Exploration: Calculated death percentages, infection rates, and continent-wide trends.
  • Tableau Dashboard: Visualized cases, deaths, and infection trends using maps, bar charts, and line graphs.

Key Insights:

  • Global Cases: 150.57M | Deaths: 3.18M | Death Rate: 2.11%.
  • Highest Deaths: Europe (1.016M) | Lowest: Oceania (1,046).
  • Highest Infection Rate: Andorra (16.71%).
